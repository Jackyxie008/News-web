# 页面设计说明（Desktop-first）

## 全局设计（适用于所有页面）

### Layout
- 采用桌面优先三栏布局：左侧“新闻列表”，中间“世界地图”，右侧“饼图 + 折线图”。
- 整体使用 CSS Grid 为主（3 列），内部模块使用 Flexbox 排版。
- 断点建议：
  - ≥1280px：三栏固定，地图为主视觉（中间列最大）。
  - 1024–1279px：三栏压缩，列表与右侧图表可折叠为抽屉（保持地图可用）。
  - <1024px：降级为上下结构（地图在上，列表与图表为 Tab/折叠面板）。

### Meta Information
- Home
  - title：新闻世界地图｜全球新闻分布与趋势
  - description：通过地图与图表快速浏览新闻的地域分布与时间趋势。
  - og:title / og:description 与 title/description 一致
- News Detail
  - title：{新闻标题}｜新闻世界地图
  - description：{新闻摘要或前 120 字}

### Global Styles（设计令牌/外观约束）
- 背景：浅色底（如 #F7F8FA）+ 卡片白底（#FFFFFF），与 Figma 组件一致。
- 字体：无衬线；标题/正文/辅助信息三档字号。
- 颜色：
  - 主色：用于高亮选中态、按钮主操作
  - 强调色：用于图表高亮、Marker 选中
  - 中性色：文本与分割线
- 交互状态：
  - Hover：卡片轻阴影/边框加深
  - Selected：列表项与 Marker 同步高亮（颜色一致）
  - Disabled：置灰且不可点击
- 动效：仅允许轻量过渡（150–250ms），禁止大面积动画导致掉帧。

## 主页（地图 + 新闻列表 + 图表）

### Layout
- 页面根容器：CSS Grid（columns: 360px 1fr 360px；gap 16–20px；全高视口）。
- 左栏与右栏为“可滚动面板”，中间地图区域固定占满剩余空间。
- 面板滚动：仅面板内部滚动（overflow-y: auto），避免整页滚动影响地图交互。

### Page Structure
1. 顶部（可选轻量页头）：站点标题 + 简短筛选摘要（例如“过去 7 天｜全部地区”）。
2. 主体三栏：
   - 左：新闻列表（含筛选/搜索）
   - 中：世界地图 + Marker/Popup
   - 右：饼图（上）+ 折线图（下）+ 图例/筛选摘要

### Sections & Components

#### A. 左侧：新闻列表面板
- 顶部工具区（与 Figma 组件一致）：
  - 搜索框：关键词输入；回车或停止输入后触发查询（带 debounce）。
  - 时间范围：快捷选项（24h/7d/30d）或日期范围选择（如 Figma 有）。
  - 清除筛选按钮：一键恢复默认。
- 列表区：
  - 列表项卡片字段：标题（1–2 行截断）、时间、来源、地点（国家/城市）。
  - 选中态：边框/底色高亮；左侧有色条或图标指示。
  - 交互：
    - 点击列表项：更新“当前选中新闻 selectedNewsId”。
    - 选中后联动：地图定位（flyTo/fitBounds）→ 对应 Marker 高亮 → 打开 Popup。
  - 空状态：提示“暂无结果”，提供“清除筛选”。
  - Loading：骨架屏；不得阻塞地图交互。

#### B. 中间：世界地图区域
- 地图容器：全高卡片/无边框全 bleed（按 Figma）；左上角放置图层/缩放控件（如组件有）。
- Marker 视觉：
  - 默认：与主题色一致的小圆点/图钉；密集时显示聚合（Cluster）。
  - Hover：放大/描边。
  - Selected：颜色更强 + 外圈光晕；与左侧列表选中态颜色一致。
- Popup（弹窗）：
  - 内容：标题、时间、来源、地点、摘要（2–3 行）。
  - 操作：主按钮“查看详情”；次按钮“关闭”。
  - 关闭方式：点击关闭按钮/点击地图空白处。
- 地图交互：
  - 拖拽/缩放：默认启用。
  - 点击 Marker：设置 selectedNewsId，并触发左侧列表滚动定位到对应项。

#### C. 右侧：图表面板（饼图 + 折线）
- 顶部：饼图卡片
  - 标题：如“地区分布”/“来源分布”（以 Figma 定义为准）。
  - 图例：显示分类名称与占比。
  - 交互：点击扇区 = 追加/替换筛选条件（例如 region/source = X），并联动刷新列表与地图。
  - 状态：
    - Hover：显示 tooltip（分类、数量、占比）。
    - Selected：扇区高亮，其余降饱和。
- 底部：折线图卡片
  - 标题：如“新闻趋势”。
  - X 轴：时间（天/周）；Y 轴：数量。
  - 交互：
    - Hover：显示 tooltip（时间点 + 数量）。
    - 区间选择：若 Figma 有 brush，则拖拽选择时间区间并更新筛选；若无则点击两端点形成范围。

### 核心联动规则（必须实现）
- 单一选中源：selectedNewsId。
- 列表点击：selectedNewsId 更新 → 地图定位到对应坐标 → Marker 进入 selected 状态 → Popup 打开。
- Marker 点击：selectedNewsId 更新 → 左侧虚拟列表滚动到该项并高亮 → Popup 打开。
- Popup“查看详情”：进入 /news/:id。
- 饼图点击：更新 FilterState → 重新计算派生数据（列表、markers、series）。
- 折线区间：更新 FilterState.timeRange → 同步刷新。

## 新闻详情页

### Layout
- 两栏布局：左侧正文（主），右侧信息卡与小地图（辅）；在窄屏下上下堆叠。

### Page Structure
1. 顶部：返回按钮 + 标题
2. 主体：
   - 左：正文/摘要
   - 右：元信息卡（来源、时间、地点）+ 单点地图（只读）

### Sections & Components
- 正文区：
  - 标题、来源、时间、地点标签
  - 正文（或详细摘要）
- 右侧信息卡：
  - 字段：来源链接（如有）、发布时间、国家/城市、坐标
- 小地图：
  - 仅显示一个 Marker（该新闻地点），默认打开一个轻量提示。
- 返回：返回主页并尽量保留筛选条件与滚动位置（若路由状态支持）。
